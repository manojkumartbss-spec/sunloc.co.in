# How to configure in Railway Dashboard:
# 1. Click your "sunloc.co.in" service
# 2. Click Settings (gear icon)
# 3. Scroll to "Builder" section
# 4. Select "Dockerfile" as builder type
# 5. Dockerfile path: ./Dockerfile (already set by default)
# 6. Click "Deploy" button
# 7. Wait for build to complete
